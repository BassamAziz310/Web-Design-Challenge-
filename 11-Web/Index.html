Summary: Latitude vs. X

The purpose of this project was to analyze weather as you get closer to the equator. To accomplish this analysis
we first pulled from OpenWeatherMap API to assemble a dataset from over 500 cities. After assembling the dataset,
we used MatPlotLib to plot various aspects of the weather vs. latitude. Factors we have looked at included temprature,
cloudiness, windspeed and humidity. This site provides the source data and visualizations created as part of the
analysis, as well as the explanations and descriptions of any trends and correlations witnessed.
